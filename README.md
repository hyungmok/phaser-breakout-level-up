# Phaser Breakout - Level Up!

A simple Breakout-style game built with Phaser 3 that introduces a leveling system. As you clear levels, the game gets progressively harder.

## Gameplay Features

-   **Classic Breakout Action**: Control a paddle to bounce a ball and destroy bricks.
-   **Level Progression**: When all bricks are cleared, you advance to the next level.
-   **Increasing Difficulty**: The ball's speed increases with each new level, making the game more challenging.
-   **Scoring System**: Earn points for each brick you destroy and get a bonus for clearing a level.

## How to Play

1.  Clone or download this repository.
2.  Open the `index.html` file in your web browser.
3.  Move the mouse left and right to control the paddle.
4.  Don't let the ball fall off the bottom of the screen!

## Built With

Enjoy the game!
